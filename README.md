## Краткое описание проекта

Проект посвящён автоматизированному подбору параметров двухконтурного PID-регулятора для системы «тележка — перевёрнутый маятник». Решение объединяет нелинейную математическую модель объекта управления, набор метрик качества (время установления, интегральные ошибки, энергию управления, штрафы за падение маятника) и несколько метаэвристических алгоритмов оптимизации.

**Ключевые особенности проекта:**

* Идентичная модель маятника для всех методик, включая насыщение усилия в интегральных звеньях
* Многофакторная функция приспособленности с контролем устойчивости и запасов по углу/позиции
* Реализация четырёх стратегий оптимизации: Artificial Bee Colony (ABC), Particle Swarm Optimization (PSO), Genetic Algorithm (GA) и Gravitational Search Algorithm (GSA), Shepherd Optimization Algorithm (SOA)
* Визуализация: графики сходимости, временные диаграммы, фазовые портреты, покадровая диагностика, анимации движения

## Технологический стек проекта

+ Язык: Python 3.10+
+ Среды: Jupyter Notebook / JupyterLab, IPython
+ Библиотеки: NumPy, SciPy, Matplotlib, ipywidgets, imageio
